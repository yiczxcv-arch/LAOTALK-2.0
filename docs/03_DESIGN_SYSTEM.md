# 16. 최종 원칙

모든 화면은 다음 기준을 만족해야 한다.

- 모바일에서 가장 먼저 검토한다.
- 사용자는 고민 없이 예약 문의까지 이동할 수 있어야 한다.
- 콘텐츠가 브랜드를 돋보이게 해야 한다.
- 디자인은 단순하고 일관성을 유지한다.
