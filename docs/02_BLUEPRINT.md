# 02_BLUEPRINT.md

# LAOTALK 2.0 BLUEPRINT

Version 1.0

---

# 목적

이 문서는 LAOTALK 2.0의 전체 홈페이지 구조, 사용자 흐름, 관리자 구조, 데이터 흐름을 정의한다.

개발자는 본 문서를 기준으로 기능을 구현하며, 설계 완료 전에는 코드를 작성하지 않는다.
