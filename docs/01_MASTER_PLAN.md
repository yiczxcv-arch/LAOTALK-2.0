# LAOTALK 2.0 MASTER PLAN

Version 1.0

---

# 1. 프로젝트 개요

## 프로젝트명
**LAOTALK 2.0**

## 프로젝트 비전
LAOTALK 2.0은 라오스를 가장 쉽고 신뢰할 수 있게 소개하는 콘텐츠 기반 여행 브랜드를 구축하는 프로젝트이다.

우리는 단순히 여행 상품을 판매하는 것이 아니라, 현지에서 직접 제작한 콘텐츠를 통해 신뢰를 쌓고, 여행을 계획하는 사람들이 자연스럽게 예약 상담까지 이어질 수 있는 구조를 만든다.

---

# 2. 프로젝트 미션

- 라오스를 처음 접하는 사람도 쉽고 편하게 여행을 준비할 수 있도록 돕는다.
- 현지에서 직접 제작한 콘텐츠를 통해 정확하고 신뢰할 수 있는 정보를 제공한다.
- 검증된 현지 파트너와 연결하여 만족도 높은 여행 경험을 만든다.

---

# 3. 프로젝트 핵심 철학

콘텐츠 → 브랜드 → 신뢰 → 예약 문의 → 현지 운영 → 좋은 여행 경험 → 후기 → 새로운 콘텐츠

이 순환 구조를 LAOTALK의 핵심 성장 모델로 삼는다.

---

# 4. 브랜드 정의

## 우리는 누구인가

LAOTALK는 라오스 여행 콘텐츠 브랜드이다.

우리는 여행 정보를 전달하고, 여행을 쉽게 준비하도록 돕고, 예약 상담을 연결하는 역할을 한다.

현지 운영은 파트너가 담당한다.

## 우리는 하지 않는 것

- 홈페이지 직접 결제
- 회원가입 중심 서비스
- 광고만을 위한 플랫폼
- 복잡한 커뮤니티 중심 운영
